# Coding-Sprint
Overview
An interactive text-based adventure game where you create a character and explore multiple locations, making choices that affect your health, score, and inventory.

Command	Description
explore -	Visit the dark forest and encounter a wild creature
cave -	Enter a mysterious cave and find a treasure chest
village -	Visit a peaceful village and help (or ignore) a villager
castle - Approach a grand castle and deal with guards
encounter -	Trigger a random encounter (merchant or magical spring)
status - View your current health, score, and inventory
quit -	Exit the game
